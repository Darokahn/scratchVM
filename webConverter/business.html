Stem Center Affiliate Program

Learn Computer Architecture & Build Your Own Console

Kids can sign up for a class at the Stem Center to learn about computer architecture while building a ScratchVM game console. The class covers embedded systems, programming, and hardware integration.

Students learn about microcontrollers, display systems, input devices, and how these components work together. By the end of the class, students will have built their own console.
